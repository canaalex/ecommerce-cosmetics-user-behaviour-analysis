# E-commerce Cosmetics User Behaviour Analysis

This project analyzes user interaction data from an online cosmetics platform to understand how users move from browsing to purchase.

## Key Insights

- Most users view products but do not complete purchases
- Users often add items to cart but drop off before buying
- Most purchases happen within a short time (within an hour)
- Repeat users are much more likely to purchase
- Users tend to view higher-priced products but buy cheaper ones
