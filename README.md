# E-commerce User Behaviour Analysis

This project analyzes user interaction data from an online cosmetics platform to understand how users move through the purchase journey.

## Key Insights

- Most users view products, but only a small percentage complete purchases
- Users often add items to cart but drop off before purchasing
- Most purchases happen within a short time (often within an hour)
- Repeat users are significantly more likely to convert
- Users tend to view higher-priced products but purchase cheaper ones

## Tools Used

- Python
- Pandas
- Matplotlib

## Project Files

- `analysis.ipynb` – main analysis notebook
- PDF report – summary of findings

## Dataset

Kaggle: eCommerce Events History in Cosmetics Shop
